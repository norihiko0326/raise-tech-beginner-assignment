## 4.1 ホーム画面（/）

**レイアウト構成：**
```
┌────────────────────────────────────────┐
│ 家計簿アプリ                    [メニュー]│
├────────────────────────────────────────┤
│ 2026年6月                            │
│ ┌──────────────────────────────────┐ │
│ │ サマリー                         │ │
│ │ 収入      ¥200,000              │ │
│ │ 支出      ¥150,000              │ │
│ │ 残高      ¥50,000               │ │
│ └──────────────────────────────────┘ │
│                                    │
│ [◀ 5月]  6月  [7月 ▶]           │
│                                    │
│ ┌──────────────────────────────────┐ │
│ │ 📊 グラフを見る                  │ │
│ │ ➕ 新しく記録                    │ │
│ │ 🔧 カテゴリ管理                  │ │
│ └──────────────────────────────────┘ │
│                                    │
│ 記録一覧（6月）                    │
│ ──────────────────────────────────│ │
│ 6/15（日）  🍔 食費      -¥1,500   │ │
│ 6/14（土）  給料         +¥150,000 │ │
│ 6/10（月）  🚌 交通費    -¥2,000   │ │
│ 6/05（水）  💡 光熱費    -¥8,500   │ │
│ ...                            │ │
└────────────────────────────────────────┘
```

**コンポーネント仕様：**

| コンポーネント | 詳細 |
|---|---|
| **月選択エリア** | 前月・次月ボタン + 当月表示。クリックで月が変わり、サマリーと記録一覧が自動更新 |
| **サマリーボックス** | 収入・支出・残高を大きく表示。背景は薄い背色 |
| **記録一覧** | 日付降順（新しい順）でソート。スクロール可能（高さ制限あり） |
| **記録アイテム** | 日付 + 曜日 + アイコン + カテゴリ名 + 金額。支出は赤、収入は青 |
| **ボタン群** | グラフボタン → グラフ画面へ。新規記録ボタン → 新規記録ダイアログ。カテゴリ管理ボタン → カテゴリ管理画面へ |

**操作仕様：**
- 記録一覧の各行をクリック → 編集ダイアログを開く
- 「◀ / ▶」ボタン → 月を遡る（最大6ヶ月前まで）
- 「新しく記録」ボタン → 新規記録ダイアログを開く（日付は今日）

## 4.2 新規記録ダイアログ

**レイアウト：**
```
┌────────────────────────────────┐
│ 新しく記録           [×閉じる] │
├────────────────────────────────┤
│ 日付 *                        │
│ [2026-06-15 ▼]               │
│                              │
│ 金額 *                        │
│ [        ] 円                 │
│                              │
│ 種類 *                        │
│ ◉ 支出  ○ 収入              │
│                              │
│ カテゴリ *                    │
│ [食費 ▼]                     │
│                              │
│ メモ（任意）                  │
│ [                      ]      │
│                              │
│ [保存]  [キャンセル]          │
└────────────────────────────────┘
```

**入力項目仕様：**

| 項目 | 型 | 必須 | 制約・バリデーション | エラーメッセージ |
|---|---|---|---|---|
| **日付** | DATE | ○ | 過去6ヶ月 ～ 今日 | 「6ヶ月以上前の日付は入力できません」 |
| **金額** | INTEGER | ○ | 1 ～ 9,999,999 | 「1円以上9,999,999円以下で入力してください」 |
| **種類** | RADIO | ○ | 支出 or 収入 | （選択必須） |
| **カテゴリ** | SELECT | ○ | 既存カテゴリから選択 | （ドロップダウンから選択） |
| **メモ** | TEXT | × | 最大255文字 | 「255文字以内で入力してください」 |

**操作仕様：**
- 「保存」クリック → バリデーション実行 → API POST → ホーム画面に戻り記録一覧更新
- 「キャンセル」または「×」 → 入力内容は破棄してダイアログを閉じる
- 編集時は既存データを入力欄に反映し、「保存」は PUT リクエスト送信

## 4.3 グラフ画面（/chart）

**レイアウト：**
```
┌────────────────────────────────┐
│ 月別収支グラフ          [戻る]  │
├────────────────────────────────┤
│ 過去6ヶ月の推移                │
│                              │
│        金額（円）              │
│        ↑                     │
│  200万┤    ┌──┐             │
│       │    │  │     ┌──┐    │
│  150万┤    │  │ ┌──┐│  │   │
│       │    │  │ │  ││  │   │
│  100万┤    │  │ │  ││  │   │
│       │    │  │ │  ││  │   │
│   50万┤ ┌──┤  │ │  ││  │   │
│       │ │  │  │ │  ││  │   │
│    0 ├─┴──┴──┴─┴──┴┴──┴─→ │
│       1月 2月 3月 4月 5月 6月│
│                              │
│ ■ 収入   ■ 支出              │
└────────────────────────────────┘
```

**グラフ仕様：**
- グラフライブラリ：recharts
- グラフタイプ：縦棒グラフ（月ごとに収入・支出を並べて表示）
- 表示対象：過去6ヶ月分（当月含む）
- X軸：月（1月 ～ 6月）
- Y軸：金額（円、0から自動スケーリング）
- 凡例：収入（青）、支出（赤）
- ツールチップ：マウスホバーで金額を表示

**操作仕様：**
- 「戻る」ボタン → ホーム画面に戻る
- グラフはスクロール・ズーム不可（固定表示）

## 4.4 カテゴリ管理画面（/categories）

**レイアウト：**
```
┌────────────────────────────────┐
│ カテゴリ管理              [戻る]│
├────────────────────────────────┤
│ 初期カテゴリ（編集不可）       │
│ ──────────────────────────── │
│ ✓ 🍔 食費                    │
│ ✓ 🚌 交通費                  │
│ ✓ 🎬 娯楽費                  │
│ ✓ 💊 医療費                  │
│ ✓ 👕 衣服費                  │
│ ✓ 🏠 家賃・住宅              │
│ ✓ 💡 光熱費                  │
│ ✓ 📱 通信費                  │
│ ✓ 💼 仕事関連                │
│ ✓ 🎁 その他                  │
│                              │
│ ユーザー作成カテゴリ           │
│ ──────────────────────────── │
│ 🎓 教育費        [削除]       │
│ 🐶 ペット費      [削除]       │
│ 🎮 ゲーム・趣味  [削除]       │
│ ...                         │
│                              │
│ [➕ 新規カテゴリ追加]         │
└────────────────────────────────┘
```

**コンポーネント仕様：**

| セクション | 詳細 |
|---|---|
| **初期カテゴリ** | チェックマーク付きで表示（全て有効）。削除ボタンなし |
| **ユーザー作成カテゴリ** | アイコン + 名前 + 削除ボタン。削除されると記録からも参照不可（注：既存の記録のカテゴリは保持） |
| **新規追加ボタン** | クリックするとダイアログを開く |

**新規カテゴリ追加ダイアログ：**
```
┌────────────────────────────────┐
│ 新規カテゴリ         [×閉じる]  │
├────────────────────────────────┤
│ カテゴリ名 *                   │
│ [              ]               │
│                              │
│ アイコン *                     │
│ [👍 ▼]  （絵文字選択）       │
│                              │
│ [追加]  [キャンセル]          │
└────────────────────────────────┘
```

| 項目 | 型 | 必須 | 制約 | エラーメッセージ |
|---|---|---|---|---|
| **カテゴリ名** | VARCHAR | ○ | 1 ～ 50文字。重複不可 | 「カテゴリ名は1～50文字、既に存在しないことを確認してください」 |
| **アイコン** | EMOJI | ○ | 絵文字1文字 | （ドロップダウンから選択） |

**操作仕様：**
- 「追加」クリック → バリデーション → API POST → 画面更新
- 「削除」クリック → 確認ダイアログ表示 → 確認後 API DELETE → 画面更新
- 「戻る」ボタン → ホーム画面に戻る

## 4.6 共通仕様

**ナビゲーション：**
- ホーム画面が主画面。他のすべての画面から戻れる
- 「戻る」ボタンはすべての副画面（グラフ画面、カテゴリ管理画面）に配置
- ダイアログはモーダル（背景をクリックしても閉じない、キャンセルボタンで閉じる）

**レスポンシブ対応：**
- PCブラウザのみ対応（1024px 以上推奨）
- モバイル対応なし（タッチ操作未対応）

**色設計：**
- 支出：赤（#E53935）
- 収入：青（#1976D2）
- 初期カテゴリ：グレー（無効化）
- ボタン：主色（#1976D2）

**フォント・サイズ：**
- タイトル：16px，太字
- ラベル：14px
- 入力欄：14px
- 記録アイテム：13px

**操作フィードバック：**
- 保存成功時：トースト通知「保存しました」（2秒表示）
- 削除成功時：トースト通知「削除しました」（2秒表示）
- バリデーションエラー：入力欄下に赤いメッセージ表示
