# 制約条件・非機能要件

**[← REQUIREMENTS.md に戻る](../REQUIREMENTS.md)**

---

## 対応デバイス・ブラウザ

| 項目 | 要件 |
|------|------|
| **対応デバイス** | PC ブラウザのみ（スマートフォン・タブレット非対応） |
| **推奨解像度** | 1024px 以上 |
| **ブラウザ対応** | Chrome, Firefox, Safari 最新版 |
| **ブラウザバージョン** | ES2020 対応版以上 |

---

## パフォーマンス要件

| 項目 | 要件 | 備考 |
|------|------|------|
| **画面表示時間** | 3秒以内 | 初期読み込み時 |
| **API 応答時間** | 1秒以内 | 通常時 |
| **データベースクエリ** | 100ms 以内 | 月別取得時 |
| **グラフレンダリング** | 500ms 以内 | recharts でのレンダリング |

---

## データ保持期間

| 項目 | 要件 |
|------|------|
| **データ保有期間** | 制限なし（全データ永続保存） |
| **表示期間** | 過去6ヶ月間は月別に表示可能 |
| **バックアップ保持** | 7日間 |
| **ログ保持期間** | 90日間 |

---

## 対応デバイス・機能の制限

| 項目 | サポート状況 |
|------|------------|
| **PC ブラウザ** | ✅ 完全サポート |
| **スマートフォン** | ❌ サポートなし |
| **タブレット** | ❌ サポートなし |
| **ネイティブアプリ** | ❌ スコープ外 |
| **オフライン機能** | ❌ スコープ外 |

---

## API 仕様の制約

| 項目 | 制約 |
|------|------|
| **リクエスト形式** | JSON（Content-Type: application/json） |
| **レスポンス形式** | JSON |
| **認証** | なし（1ユーザー専用） |
| **レート制限** | 1分間に100リクエスト/IP |
| **タイムアウト** | 30秒 |

---

## ブラウザ機能の制約

| 機能 | サポート状況 | 備考 |
|------|------------|------|
| **LocalStorage** | ✅ 使用可 | セッション情報等に使用 |
| **Cookie** | ✅ 使用可 | CSRF トークン等に使用 |
| **IndexedDB** | ❌ 不要 | スコープ外 |
| **ServiceWorker** | ❌ 不要 | オフライン機能なし |

---

## データベース制約

| 項目 | 制約 |
|------|------|
| **文字コード** | UTF-8MB4（絵文字対応） |
| **同時接続数** | 10 同時接続（ローカル開発） |
| **データサイズ上限** | なし（AWS RDS の制限に準ずる） |
| **トランザクション** | ACID 準拠 |

---

## 実装上の制約

| 項目 | 制約 |
|------|------|
| **外部 API 連携** | なし |
| **メール送信機能** | なし |
| **ファイルアップロード** | なし |
| **リアルタイム更新** | なし（定期的なポーリング不要） |
| **プッシュ通知** | なし |

---

## セキュリティ制約

| 項目 | 制約 |
|------|------|
| **認証方式** | IP ホワイトリスト（本番環境） |
| **ユーザー認証** | なし |
| **暗号化** | HTTPS（本番環境） |
| **セッション管理** | Cookie ベース（スコープ外） |

---

## ユーザーインターフェース制約

| 項目 | 制約 |
|------|------|
| **言語** | 日本語のみ |
| **多言語対応** | スコープ外 |
| **ダークモード** | スコープ外 |
| **アクセシビリティ** | 基本的な WCAG 2.1 AA 対応（スコープ外） |

---

## スケーラビリティ制約

| 項目 | 制約 | 目安 |
|------|------|------|
| **ユーザー数** | 1ユーザー | 複数ユーザー対応は将来版で実装 |
| **月別データ容量** | 制限なし | 1月あたり 1000件 の記録想定 |
| **同時ユーザー数** | 1 | 複数アクセス非対応 |

---

## 今後のための予約機能（スコープ外）

以下の機能は将来的に追加予定だが、現バージョンでは実装しない：

- ユーザー認証機能（複数ユーザー対応）
- CSVエクスポート/インポート機能
- 予算設定・アラート機能
- レシート画像からの自動取得
- 家族間での共有機能
- モバイル対応（レスポンシブ）
- ダークモード
- 複数通貨対応
- 定期的な取引（サブスクリプション）記録
