# 家計簿アプリ 要件定義書

**作成日：** 2026年6月15日 | **バージョン：** 1.0

---

## 📌 概要

このアプリケーションは、日々の収支を記録・管理するためのWebアプリケーションです。
ユーザーがお金の使い道をカテゴリ分けして記録し、月別に集計・確認できます。

**ログイン機能なし** — 1ユーザー専用のシンプルな設計

---

## ✨ 主な機能

| 機能 | 説明 |
|------|------|
| **収支の記録** | 日付、金額、カテゴリ、メモを入力して保存 |
| **収支の編集** | 保存済みの記録を後から修正可能 |
| **収支の削除** | 不要な記録を削除可能 |
| **カテゴリ管理** | 王道カテゴリを事前用意 + ユーザーが新しいカテゴリを追加可能 |
| **月別集計** | 月ごとに収入合計・支出合計を表示 |
| **グラフ表示** | 月別の収支をグラフで可視化 |
| **過去データ閲覧** | 過去6ヶ月分のデータを月タップで閲覧可能 |
